## Présentation

Ce repository héberge le guide de déploiement technique de MyCarbone, extension de mesure d'empreinte carbone pour requêtes IA, à destination des organisations.

**Public cible :**
- Administrateurs systèmes
- DSI (Directions des Systèmes d'Information)
- Responsables postes de travail
- Équipes IT

## Contenu du guide

Le guide de déploiement couvre les aspects suivants :

### 1. Prérequis techniques
- Navigateurs supportés (Chrome, Edge, Brave, Firefox)
- Permissions requises
- Compatibilité systèmes d'exploitation

### 2. Méthodes de déploiement
- **Google Workspace** (Gestion centralisée)
- **Microsoft Intune** (Endpoint Manager)
- **GPO Active Directory** (Stratégies de groupe)
- **Firefox ESR** (Policies.json)
- Déploiement hors ligne / air gap

### 3. Configuration centralisée
- Paramètres par défaut
- Scripts d'injection
- Personnalisation entreprise

### 4. Sécurité et conformité
- Analyse des risques
- Conformité RGPD
- Architecture 100% locale

### 5. Support prioritaire
- Contact direct équipe entreprise
- Délais d'intervention garantis
