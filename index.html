PRINCIPE FONDAMENTAL : MyCarbone ne collecte aucune donnée personnelle et ne communique avec aucun serveur externe. Architecture 100% locale, aucune infrastructure cloud requise.

1. Prérequis techniques

Navigateurs supportés

NavigateurVersion minimum
Google Chrome88+ (Manifest V3)
Microsoft Edge88+
Brave1.20+
Firefox91+
SafariNon supporté

Permissions requises

  • storage — Stockage local des statistiques
  • webRequest — Interception des requêtes IA
  • notifications — Alertes budget (optionnel)
  • downloads — Génération des rapports PDF
  • activeTab — Accès à l'onglet actif

2. Déploiement Chrome / Edge / Brave

Google Workspace

  1. Connectez-vous à admin.google.com
  2. Appareils > Chrome > Applications et extensions
  3. Ajouter l'extension via l'ID Chrome Web Store
  4. Forcer l'installation sur les unités d'organisation cibles

Microsoft Intune

ExtensionInstallForcelist = [
  "ID_EXTENSION;https://clients2.google.com/service/update2/crx"
]

GPO Active Directory

Configuration ordinateur > Modèles d'administration > Google Chrome > Extensions > Configurer la liste des extensions imposées

3. Déploiement Firefox

Fichier policies.json à placer dans le dossier d'installation :

{
  "policies": {
    "Extensions": {
      "Install": ["https://votre-serveur/mycarbone.xpi"],
      "Locked": ["mycarbone@extension"]
    }
  }
}

4. Configuration centralisée

Script d'injection de configuration par défaut :

chrome.storage.local.set({
  budget: { enabled: true, daily: 50 },
  lastResetDate: new Date().toDateString()
});

5. Sécurité et conformité

  • Aucune connexion réseau sortante
  • Aucun stockage cloud
  • Aucun accès au contenu des pages
  • Code source auditable sur demande
  • Conforme RGPD par conception
